Core Modules

Six Core Modules

These six modules form the core of the overall system. Each serves a different function and can also be flexibly combined depending on project needs.

Virtual Production

UE Virtual

A virtual studio and real-time scene engine that supports green screen production, virtual scene switching, and more advanced content presentation.

Show Control

U-Show

The core for program output and broadcast control, supporting multi-camera switching, workflow management, and overall program operation.

Graphics

U-CG

A graphics and captioning system that supports titles, lower thirds, scoreboards, and a wide range of visual overlay applications.

Teleprompter

U-TP

A teleprompting system that helps hosts, presenters, and guests deliver live presentations, speeches, and course recordings more smoothly and confidently.

Signal Converter

U-Hub

A three-way signal converter used for conversion and integration between different signal formats, improving system compatibility and on-site stability.

Digital Human

U-Gemini

A digital human system for AI presenter content, virtual character applications, and automated content expansion.
